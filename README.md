# Python
Python是机器学习中使用最多的编程语言之一,学习过程中有一些模块和语句十分常见,这里我根据看的一些教程总结了一下numpy、pandas、matplotlib的常见用法,以便于未来查找,节省时间。
## 模块安装
- __常规__  
pip install 模块名
- __Anaconda__  
conda install 模块名
